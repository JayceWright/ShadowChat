# Shadow-Net Chat (WebSocket)

## Backend (Java)
- Работает на порту `8080`.
- Использует стандартные библиотеки Java (или `Java-WebSocket`, если базовых не хватит).
- Логика: Принимает соединения на эндпоинт `ws://localhost:8080/chat`. При получении текстового сообщения от любого клиента рассылает его всем активным сессиям.

## Frontend (Vanilla JS)
- Папка `/frontend` (index.html, app.js, style.css).
- Дизайн: Темный хакерский терминал (черный фон, зеленый моноширинный шрифт).
- Логика: Устанавливает соединение `new WebSocket('ws://localhost:8080/chat')`. Имеет инпут для ввода и окно логов. При получении сообщения выводит его на экран.
